03

Orquestación Proxy Inverso SSL

Configuración del túnel seguro para flask.jocarsa.com. Apache actúa como terminador SSL y delega el tráfico a la instancia local.

  • # a2enmod proxy proxy_http
  • # a2enmod headers ssl
  • # systemctl restart apache2
flask.jocarsa.com.conf 
<VirtualHost *:443>
    ServerName flask.jocarsa.com
    SSLEngine on
    SSLCertificateFile /etc/apache2/ssl/jocarsa_combined.cer

    ProxyPreserveHost On
    ProxyPass / http://127.0.0.1:5000/
    ProxyPassReverse / http://127.0.0.1:5000/
</VirtualHost>
